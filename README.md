# Sales Data Analysis Dashboard

## Project Overview
This project features an **Interactive Sales Dashboard** created using Microsoft Excel. The goal was to analyze sales performance and provide actionable insights through data visualization.

## Key Features
* **Dynamic Visuals:** Used various charts (Bar, Line, and Map charts) to represent data.
* **Data Cleaning:** Processed raw sales data to ensure accuracy.
* **Insights:** Identified top-performing regions and sales trends.
